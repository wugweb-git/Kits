# PATTERN REGISTRY

Document recurring patterns with confidence.

## Purpose
This file stores repeated visual findings that have been observed across Wugweb references.

## Example pattern format
- Pattern name
- Evidence sources
- Meaning
- Confidence
- Rule

## Early approved patterns
- Collage-first composition
- Designer-made editorial feel
- Rounded geometry dominance
- One dominant geometric anchor
- Real UI over fake widgets
- Limited palette per artwork
- Asymmetrical editorial balance
- Metaphor-led visuals

## Rule
Only promote a pattern here if it is repeated across multiple sources.
