# Wugweb Brand OS

Evidence-driven Brand OS for Wugweb.

This folder is the foundation of an Obsidian-style knowledge base. It captures the core philosophy, vocabulary, decisions, and reference logic that will power Wugweb-aligned marketing, brand, and AI-generated visuals.

## Working approach
Every new visual or brand decision should pass through this chain:

Reference → Observation → Pattern → Principle → Rule → Prompt → Artwork

## Current direction
- Collage-first, not illustration-first.
- Designer-made, not AI-made.
- Simplification wins.
- Rounded geometry should dominate.
- Prefer objects over dashboards.
- Use one dominant geometric anchor.
- Keep the palette limited per artwork.
- Use asymmetrical editorial balance.
- Communicate through metaphor, not text.
