# VISUAL REASONING

This file explains what visual primitives mean in Wugweb language.

## Core mappings
- Circles = ecosystems, markets, loops, attention fields
- Arrows = transformation, progress, orchestration, movement
- Grids = architecture, structure, systems, decision spaces
- Cards = products, modules, capabilities, information blocks
- Rounded containers = calmness, premium quality, human-centered design
- Negative space = clarity, hierarchy, editorial balance

## Working rule
Never use a shape only because it looks nice. Every shape must carry meaning.
