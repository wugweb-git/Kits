# OBSERVATIONS

This file captures repeated evidence from the Wugweb reference set.

## Analysis chain
Reference → Observation → Pattern → Principle → Rule

## Key observations
- The strongest visuals are collage-first rather than illustration-first.
- The work feels designer-made, not AI-generated.
- Simplification consistently improves the result.
- Rounded geometry is more aligned with the brand than sharp technical forms.
- Real UI elements feel stronger than fake widgets.
- One dominant anchor creates better hierarchy than multiple competing focal points.
- Limited palettes make the compositions feel more premium.
- Asymmetrical editorial balance feels more intentional than centered layouts.
- Conceptual metaphors outperform literal explanations.
- Dense node graphs and generic AI-brain motifs weaken the brand.

## Rule of thumb
If a visual pattern appears once, treat it as a candidate. If it appears repeatedly across sources, promote it.
