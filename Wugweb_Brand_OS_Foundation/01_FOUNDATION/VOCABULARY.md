# Vocabulary

This vocabulary defines the language Wugweb uses internally when describing systems, visuals and strategy.

## Core terms
- Transformation
- Architecture
- Signals
- Networks
- Blueprint
- Orchestration
- Decision
- System
- Capability
- Structure
- Flow
- Clarity
- Metaphor
- Composition

## Usage rule
Use words that describe systems and relationships, not decorative or hype-driven language.
