# CURRENT CONTEXT

## Objective
Build an Obsidian-style Brand OS for Wugweb inside the Kits repository.

## Current direction
The visual language has converged toward collage-first, designer-made editorial compositions using cut-out photography, rounded geometry, asymmetric balance, limited palettes, and conceptual metaphors.

## Approved principles
- Collage-first, not illustration-first.
- Designer-made, not AI-made.
- Simplification wins.
- Rounded language should dominate.
- Avoid thin network lines unless central to the idea.
- Prefer objects over dashboards.
- Use one dominant geometric anchor.
- Keep the palette limited per artwork.
- Use asymmetrical editorial balance.
- Communicate through metaphor, not text.

## Explicit bans
- Generic glowing AI brains.
- Dense node graphs.
- Tiny floating analytics widgets.
- Dashboard-heavy compositions unless the subject is software.
- Random technology icons.
- Overly detailed illustrations.
- Gradient-heavy AI aesthetics.
- Excessive clutter.
